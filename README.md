# The Lost Horizons

## Requirements

- Python 3.10 or later
- pygame-ce
- pytmx

---

## Installation Guide

### 1. Install Python
-[Download Python here](https://www.python.org/downloads/)

### 2. Install Game

- pip install pygame-ce
- pip install pytmx


### 3. Run the Game

From the same terminal, start the game:

```bash
python main.py
